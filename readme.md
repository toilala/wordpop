# WordPop

A fast, no-nonsense dictionary for people who read a lot and want definitions without the hassle of tabs. No AI, no tracking,no nonsense definitions only .


Most dictionary extensions are bloated, slow and just don't fit my needs. I had a hard time finding an extension that was a simple, instant popup without any hassle.I built it as a convinience tool for myself.

- Uses DictionaryAPI for words, Wikipedia for entities (like "New York City"), and Google Translate as a fallback for phrases.

- Saves lookups in your browser cache. Next time you see the word, it loads in **0ms**.

- No accounts.No logins
